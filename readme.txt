# Linux User Manager

Linux User Manager is a command-line Python script designed for managing users, groups, and file permissions on Linux systems. It provides a simple and interactive interface to perform various system administration tasks such as adding or removing users, managing groups, and checking for files with risky permissions.

## Features

- List all users and groups on the system.
- Add and delete users.
- Create and remove groups.
- Add users to and remove them from groups.
- Check for files with risky permissions (e.g., 777).
- Interactive menu with option selection.

## Requirements

- Python 3.x
- Linux operating system
- Root or sudo privileges

## Installation

1. Clone this repository or download the `linux_user_manager.py` file directly:

   ```bash
   git clone https://github.com/nabilmursi/linux_user_manager.py/

2. Navigate to the directory containing the script:

$ cd linux_user_manager

3. Make the script executable:

$ chmod +x linux_user_manager.py

Usage
To run the script, use the following command:

$ sudo python3 linux_user_manager.py

Once started, the script will display a list of available commands. You can select an action by entering the corresponding number. If you need to see the list of options again, type 10.
